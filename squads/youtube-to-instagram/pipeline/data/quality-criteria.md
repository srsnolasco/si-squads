# Quality Criteria — YouTube → Instagram

## Critérios de Texto (Iago Instagram)

| Critério | Mínimo aceitável | Ideal |
|---------|-----------------|-------|
| Hook da legenda | Primeiros 125 chars provocam "ver mais" | Primeira linha para sozinha como afirmação impactante |
| Slides do carrossel | 40 palavras por slide (headline + supporting text) | 60-70 palavras, com dado ou exemplo por slide |
| Número de slides | 6 slides mínimo | 8-10 slides |
| CTA | CTA presente e específico | CTA com palavra-chave para DM ou pergunta de engajamento |
| Hashtags | 8 tags, mix de nicho e amplo | 10-12 tags variadas, sem repetição de runs anteriores |
| Tom | Consultivo, B2B, sem linguagem ao consumidor final | Parceiro estratégico, específico, com vocabulário do nicho |
| Aderência ao vídeo | Conteúdo derivado do vídeo informado | Insights diretos do vídeo, sem invenção |

## Critérios de Imagem (Diego Design)

| Critério | Mínimo aceitável | Ideal |
|---------|-----------------|-------|
| Dimensões | 1080x1440px exatos | 1080x1440px com overflow:hidden |
| Font size hero/título | 60px | 64-72px para slide de capa |
| Font size body | 34px | 36-40px |
| Font size caption | 24px | 26px |
| Contraste WCAG | 4.5:1 mínimo | > 7:1 para texto principal |
| Identidade visual | Paleta Roxo+Preto+Cinza+Branco presente | Template C Premium Dark integral |
| Consistência | Mesmo design system em todos os slides | Header + accent line + footer em todos |
| Contadores de slide | Ausentes | Ausentes (Instagram tem navegação nativa) |

## Critérios de Revisão (Renata Revisão)

### Aprovação automática (score >= 7/10 em todos)
- Relevância ao vídeo original
- Qualidade do hook e da legenda
- Estrutura e fluxo do carrossel
- Aderência à identidade visual
- Tom e voz da marca
- CTA e engajamento

### Rejeição automática (qualquer score < 4/10)
- Dado inventado (não presente no vídeo)
- Linguagem voltada ao consumidor final (não ao corretor)
- Identidade visual ausente ou errada
- Slides abaixo de 40 palavras
- Ausência de CTA

### Escalação (após 3 ciclos de revisão sem aprovação)
- Apresentar ao usuário para decisão manual
